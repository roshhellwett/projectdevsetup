# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulnerability, please open an issue at:

https://github.com/zenith-open-source/projectdevsetup/issues

We aim to acknowledge reports within 48 hours and will provide updates as we investigate and address the issue. Please do not publicly disclose the vulnerability until we have had a chance to assess and resolve it.
